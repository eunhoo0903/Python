## 02-6 집합 자료형

집합(set)은 집합에 관련된 것들을 쉽게 처리하기 위해 만든 자료형이다.

## 집합 자료형은 어떻게 만들까?

집합 자료형은 다음과 같이 set 키워드를 사용해 만들 수 있다.

```python
>>> s1 = set([1, 2, 3])
>>> s1
{1, 2, 3}
```

위와 같이 set()의 괄호 안에 리스트를 입력하여 만들거나 다음과 같이 문자열을 입력하여 만들 수도 있다.

```python
>>> s2 = set("Hello")
>>> s2
{'e', 'H', 'l', 'o'}
```

## 집합 자료형의 특징

"Hello" 문자열로 자료형을 만들었는데 생성된 자료형에는 l 문자가 하나 빠져 있고 순서도 뒤죽박죽이다. 그 이유는 다음과 같은 2가지 특징이 있다.

- 중복을 허용하지 않는다.
- 순서가 없다.

> set은 중복을 허용하지 않는 특징 때문에 데이터의 중복을 제거하기 위한 필터로 종종 사용된다.

리스트나 튜플은 순서가 있기 때문에 인덱싱을 통해 요솟값을 얻을 수 있지만, set 자료형은 순서가 없기 때문에 인덱싱을 통해 요솟값을 얻을 수 없다.

## 교집합, 합집합, 차집합 구하기

set 자료형을 정말 유용하게 사용하는 경우는 교집합, 합집합, 차집합을 구할 때 이다.

먼저 다음과 같이 2개의 set 자료형을 만들자.

```python
>>> s1 = set([1, 2, 3, 4, 5, 6])
>>> s2 = set([4, 5, 6, 7, 8, 9])
```

### 교집합 구하기

s1과 s2의 교집합을 구하려면 '&' 또는 intersection 함수를 이용해서 간단히 구할 수 있다.

```python
>>> s1 % s2
{4, 5, 6}
>>> s1.intersection(s2)
{4, 5, 6}
```

### 합집합 구하기

'|' 또는 union 함수를 이용하면 합집합을 구할 수 있다.

```python
>>> s1 | s2
{1, 2, 3, 4, 5, 6, 7, 8, 9}
>>> s1.union(s2)
{1, 2, 3, 4, 5, 6, 7, 8, 9}
```

### 차집합 구하기

-(빼기)를 사용하거나 difference 함수를 사용하면 차집합을 구할 수 있다.

```python
>>> s1 - s2
{1, 2, 3}
>>> difference(s2)
{1, 2, 3}
```

## 집합 자료형 관련 함수

### 값 1개 추가하기 - add

이미 만들어진 set 자료형에 add 함수를 사용하면 값을 추가할 수 있다.

```python
>>> s1 = set([1, 2, 3])
>>> s1.add(4)
>>> s1
{1, 2, 3, 4}
```

### 값 여러 개 추가하기 - update

여러 개의 값을 한꺼번에 추가할 때는 다음과 같이 하면 된다.

```python
>>> s1 = set([1, 2, 3])
>>> s1.update([4, 5, 6])
>>> s1
{1, 2, 3, 4, 5, 6}
```

### 특정 값 제거하기 - remove

특정 값을 제거하고 싶을 때는 다음과 같이 하면 된다.

```python
>>> s1 = set([1, 2, 3])
>>> s1.remove(2)
>>> s1
{1, 3}
```
